# 🎬 Movie Search App

A React app to search for movies and save your favourites, powered by the TMDB API.

## Features

- 🔍 Search for movies by title
- 📋 Browse movie listings
- ❤️ Save and manage your favourite movies (stored in localStorage)
- 🎥 View detailed movie information

## Tech Stack

- React
- Vite
- TMDB API

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Create a `.env` file in the root with your TMDB API key.
